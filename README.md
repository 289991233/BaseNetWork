```
 在项目 build.gradle中添加
   allprojects {
        repositories {
           jcenter()
           mavenCentral()
           maven { url "https://jitpack.io" }
       }
   }
 在App build.gradle中引用
   compile 'com.github.289991233:BaseNetWork:v0.0.8'

```
